# 基于umi和antd的文件管理程序
### 启动访问
`npm start`
访问
`http://localhost:8003/files`

### 功能
 1. 新建文件夹
 2. 上传文件 （由于测试服务器限制只能上传100K内的图片）
 3. 修改文件名
 4. 删除文件
 5. 批量移动文件
